一个简单的练习项目。
### 功能
- 产生算式，并且可以显示算式的计算结果。
- 支持选择加减乘除四种运算符。
- 支持设定计算的最大值，算式与任何中间结果都不会超过最大值。
- 这个版本只支持非负整数运算，运算中间结果当然也都是非负整数。
- 支持设定算式是否带有括号
